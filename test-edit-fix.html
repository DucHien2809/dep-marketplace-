Kiểm tra các lỗi đã được sửa trong form cập nhật sản phẩm

✅ Lỗi đã được sửa

  • ID mismatch: edit-sizes → edit-size
  • Tên cột: featured → is_featured
  • Tên cột: originalPrice → original_price
  • Thêm kiểm tra null cho tất cả form elements
  • Thêm kiểm tra bảo vệ cho file input
  • Cải thiện xử lý lỗi và thông báo

📋 Các cột cần thêm vào database

Chạy file add-missing-columns.sql trong Supabase SQL Editor để thêm các cột còn thiếu:

-- Các cột cần thêm: - price (INTEGER) - Giá bán - original_price (INTEGER) - Giá gốc - category (TEXT) - Loại sản phẩm - style (TEXT) - Phong cách - sizes (JSONB) - Kích thước - material (TEXT) - Chất liệu - origin (TEXT) - Nguồn gốc tái chế - condition (TEXT) - Tình trạng

🔍 Kiểm tra form elements

Tất cả các element trong form cập nhật đều đã được kiểm tra:

  • edit-title ✅
  • edit-category ✅
  • edit-price ✅
  • edit-original-price ✅
  • edit-size ✅
  • edit-style ✅
  • edit-material ✅
  • edit-origin ✅
  • edit-condition ✅
  • edit-featured ✅

🚀 Cách sử dụng

  1. Chạy file add-missing-columns.sql trong Supabase SQL Editor
  2. Refresh trang web
  3. Thử cập nhật một sản phẩm
  4. Kiểm tra console để xem log
